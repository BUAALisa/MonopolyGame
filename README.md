# MonopolyGame

## Part 1：游戏简介
大富翁在线游戏是一款经典的桌面游戏。
玩家在游戏开始分得游戏金钱，玩家通过掷骰子在地图上移动，购买地产、建楼、收取租金；玩家在游戏中使其他玩家破产，最后一个玩家即为胜利者。本游戏为单机游戏，提供了一个在线平台，支持双玩家。

## Part 2：游戏登录
在游戏配置界面，两名玩家可在"Player"界面分别选定头像，输入姓名并确认；在"Map"界面选定地图；在"Conditions"界面调整游戏天数、胜利资金、初始资金（默认胜利条件为出现玩家破产；默认每名玩家初始具有1000金币和100点卷作为初始资金）。点击"Start"进入游戏。

## Part 3：游戏界面与操作
1.游戏界面包括地图、玩家图标、玩家信息面板、地产、房屋、骰子等元素。
2.每回合，玩家通过鼠标点击“点击掷点”按钮抛出一颗骰子，决定移动步数；掷骰之前可选择本回合是否使用卡片。
3.当玩家到达空地或自己的平房（店铺、商场、商业大楼和摩天大楼）上，可选择支付500金币购置空地建成平房，或可选择支付1000（2000、3000、4000、5000）金币升级建筑。
4.游戏右上角显示当前游戏天数。两名玩家的资产信息（金币、点卷、房屋数量、卡片数量）分列在地图两侧。当前回合的玩家可在地图左侧查看自己持有的具体卡片。信息通报位置。

## Part 4：游戏规则与策略
1.当玩家停留在另一位玩家的建筑处，须根据房子的等级向对方支付一定的过路费。若对方该回合已入狱或住院，则免收过路费。
2.玩家经过起点可获得200金币和30点卷；玩家停留在起点可获得500金币和50点卷。
3.监狱、医院和公园
（1）当玩家停留在监狱时，有概率发生以下事件之一：“去监狱看望好友”, “被冤枉入狱”, “被监狱管理员抓去打扫卫生”。
（2）当玩家停留在医院时，有概率发生以下事件之一：“在医院检查出流感”，“碰上神经病患者”，“在医院门口遇上打劫被群殴”，“看望住院好友”。住院时间为1～4天。
（3）当玩家停留在公园时，有概率发生以下事件之一：“污水污染严重”，“随便停车被贴罚单”，“随地大小便”，“指责他人乱倒垃圾”。

4.卡片商店
当玩家停留在商店且点券数量大于0时，进入卡片商店。玩家可根据需要购置不同的卡片。卡片商店右下角显示当前玩家剩余的点卷数量。
卡片效果如下：
加盖卡：玩家当前所在的地产的建筑加盖一层。
均贫卡：玩家与另外一名玩家平分当前金币。
换屋卡：停留在有房屋的土地上时，使用此卡可交换视野内房屋。
遥控骰子：控制该玩家下一次骰子的点数。
陷害卡：使另一名玩家立刻入狱两天。
嫁祸卡：遇到危险时自动生效，将陷害（其他玩家对自己使用的负面效果的卡片）或大额过路费（大于3000金币）嫁祸给他人。
购地卡：当前位置的土地不是玩家所有时，使用此卡可以连同建筑强制购入获得此土地。
降级卡：当前位置的土地不是玩家所有时，使用此卡可以使房屋等级降一级。
抢夺卡：从另一名玩家手头随机抢夺一张卡片。
查税卡：对其他玩家收取其现金的10%的税款。
停留卡：可以对自己或其他玩家使用。使目标对象在原地停留一回合。
乌龟卡：可以对自己或其他玩家使用。使目标对象连续三回合只走一步。
卡片说明：控制步数的卡片效果会互相覆盖。

5.新闻
考验玩家运气，随机发生的事件会给玩家带来各种意想不到的局面。以下事件随机发生：
幸运事件：“获得仙女钻石戒指”、“发现宝箱”、“股市盈利”、“彩票中奖”、“家中挖到宝藏”。
不幸事件：“遭遇小偷”、“乐善好施”、“不翼而飞”、“人头被冒贷”、“过马路闯红灯”、“天降神猪”、“掉入坑中”、“乱涂乱画”、“上网被骗”、“醉酒袭警”、“遇鬼吓哭”。

## Part 5：游戏结束与结果展示
1.满足以下任意一条时，游戏结束：
（1）当前天数 > = 设定的最大游戏天数；
（2）任意一名玩家现金 > = 设定的胜利资金；
（3）任意一名玩家现金 < 0.
2.游戏结束后，出现成绩表界面，以玩家当前现金数量判定胜负。

## Part 6：游戏素材说明
该项目大部分游戏素材（包括人物角色、卡片技能、新闻事件）均来源于经典游戏“大富翁4”的抠图。房屋图标为网络素材。游戏地图为组内成员基于地产大亨原版地图进行的重绘和修改。

## Part 7：改进与不足
1.本游戏是对经典游戏“大富翁”的java复现，由于时间和工作量的关系，我们未能实现多玩家和联机功能，只基于两位玩家实现了单机游戏的基本功能。
2.由于手绘地图的精确度略有欠缺，人物的行走和停留、房屋的建设部分可能与确切位置存在偏差，但不影响游戏体验。
